The Essential GNSS Project

/apps          The applications folder:   Project files for visual C++ & linux.
/bin           The executables folder:    Fully compiled programs (win32 for now, have to build the linux apps).
/data          The data folder:           Test data sets such as raw GPS data files, etc. 
/doc           The documentation folder:  The project documentation is located in here.
/lib           The libraries folder:      Compiled binary library files are stored here.
/ref           The references folder:     When copyright is not in question, references are stored here.
/src           The 'c' source folder:     'c' source code is here.
/src_cpp       The 'cpp' source folder:   c++ source code is here.
/www           The website folder:        The project website files are stored here.
